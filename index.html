How to Use

  • Click the button to add an exam
  • Fill in the fields manually or alternatively click the clock button for a menu dialog.
  • If the exam does not have reading time you can leave it blank.
  • If you entered a reading time for the exam but wish to clear it, just select the individual time (hours, minutes, AM/PM) and click the backspace or delete key OR make it the same as the start time.
  • Click to save changes for that row. Once saved, it will turn green and a prompt will appear. You will need to do this for each row if you edit/add more than one in one go.
  • Click to remove an exam from the list.
  • If you experience any issues with this site, or want to clear all exams click the red button below.
